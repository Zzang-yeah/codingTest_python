## 1. 백준 16456번. 하와와 대학생쨩 하와이로 가는 거시와요~

https://www.acmicpc.net/problem/16456

```python
n=int(input())

dp=[1,1,2]

if n>3:
    for i in range(3, n):
        dp.append((dp[i-3]+dp[i-1])%1000000009)

print(dp[n-1])
```

```python
#문제 정확히 이해하는 데만 한시간쓴듯;이게 왜 정렬인것...?
n,m=map(int,input().split())

people=[]
for i in range(n):
    people.append(list(map(int,input().split())))

flag=True
flag1=True
flag2=True
for i in range(m):
    arr=[j[i] for j in people]
    for x in range(n-1):
        if arr[x]==1 and arr[x+1]==0:
            flag1=False
        if arr[x]==0 and arr[x+1]==0:
            flag2=False
        if flag1==flag2==False:
            flag=False
            break
    if flag==False:
        break

if flag==True:
    print('YES')
else:
    print('NO')
```
